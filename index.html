درباره ما

آموزشگاه کامپیوتر آرمان با بیش از ۱۰ سال سابقه درخشان در زمینه آموزش مهارت‌های کامپیوتری و برنامه‌نویسی، همواره در تلاش بوده تا با بهره‌گیری از اساتید مجرب و به‌روزترین متدهای آموزشی، دانشجویان را برای ورود به بازار کار آماده کند.

ماموریت ما تربیت نیروهای متخصص و کارآمد در حوزه فناوری اطلاعات است. در این راستا، تمامی دوره‌های آموزشی همراه با ارائه مدرک معتبر فنی و حرفه‌ای، جزوات تخصصی و فیلم‌های آموزشی می‌باشد.

ما معتقدیم که هر فردی با هر سطح دانشی می‌تواند برنامه‌نویسی را یاد بگیرد و در این مسیر همراه شما هستیم.

دوره جامع ICDL

مهارت‌های هفت‌گانه کامپیوتر

دوره ICDL (International Computer Driving License) استانداردی بین‌المللی برای مهارت‌های کامپیوتری است که شامل ۷ مهارت اصلی می‌شود. این دوره برای همه افراد در هر سطحی مناسب است.

مزایای دوره

  • مدرک بین‌المللی معتبر
  • یادگیری مهارت‌های ضروری کامپیوتر
  • افزایش فرصت‌های شغلی
  • مناسب برای تمام سنین

مهارت‌های آموزش داده شده

  • مفاهیم پایه فناوری اطلاعات
  • کار با سیستم عامل ویندوز
  • اینترنت و ایمیل
  • واژه‌پردازی (Word)
  • صفحات گسترده (Excel)
  • پایگاه داده (Access)
  • ارائه مطلب (PowerPoint)

سرفصل‌های دوره:

  1. مفاهیم پایه: سخت‌افزار، نرم‌افزار، شبکه‌ها
  2. ویندوز: مدیریت فایل‌ها و پوشه‌ها، تنظیمات سیستم
  3. Word: فرمت‌دهی متون، جدول‌ها، تصاویر
  4. Excel: فرمول‌نویسی، نمودارها، توابع پیشرفته
  5. Access: ایجاد پایگاه داده، جداول، کوئری‌ها
  6. PowerPoint: طراحی اسلایدها، انیمیشن‌ها
  7. اینترنت: جستجوی پیشرفته، ایمیل، امنیت

چرا ++C؟

  • سرعت اجرای بسیار بالا
  • کنترل دقیق روی سخت‌افزار
  • پایه یادگیری سایر زبان‌ها
  • استفاده در صنعت بازی‌سازی

کاربردها

  • توسعه سیستم‌عامل‌ها
  • برنامه‌نویسی سیستمی
  • بازی‌های کامپیوتری
  • برنامه‌های Embedded

سرفصل‌های دوره:

  1. مقدمات: متغیرها، عملگرها، ساختارهای کنترلی
  2. توابع: تعریف، فراخوانی، بازگشتی
  3. آرایه‌ها و اشاره‌گرها: مدیریت حافظه
  4. ساختمان داده‌ها: ساختارها، یونیون‌ها
  5. برنامه‌نویسی شی‌گرا: کلاس‌ها، وراثت
  6. فایل‌ها: خواندن و نوشتن داده‌ها
  7. پروژه پایانی: ساخت برنامه کاربردی

چرا HTML/CSS؟

  • پایه ضروری برای طراحی وب
  • یادگیری آسان و سریع
  • امکان ساخت وبسایت بدون برنامه‌نویسی
  • تقاضای بالا در بازار کار

توانایی‌های کسب شده

  • ساخت ساختار صفحات وب
  • طراحی ریسپانسیو
  • انیمیشن‌های CSS
  • بهینه‌سازی برای موتورهای جستجو

سرفصل‌های دوره:

  1. مقدمات HTML: تگ‌ها، ساختار صفحه
  2. فرم‌ها: ایجاد فرم‌های تعاملی
  3. CSS پایه: انتخابگرها، خصوصیات
  4. Flexbox و Grid: چیدمان پیشرفته
  5. رسپانسیو: Media Queries
  6. انیمیشن‌ها: ترنزیشن‌ها و کیفریم‌ها
  7. پروژه عملی: طراحی کامل یک سایت

دوره جامع برنامه‌نویسی پایتون

یادگیری آسان، بازار کار عالی

پایتون یکی از محبوب‌ترین زبان‌های برنامه‌نویسی جهان است که برای توسعه وب، هوش مصنوعی، تحلیل داده و اتوماسیون استفاده می‌شود. این زبان با سینتکس ساده و خوانا، گزینه‌ای ایده‌آل برای شروع برنامه‌نویسی است.

چرا پایتون؟

  • سینتکس ساده و خوانا
  • کتابخانه‌های قدرتمند و گسترده
  • جامعه بزرگ توسعه‌دهندگان
  • چندسکویی (Cross-platform)
  • مناسب برای پروژه‌های کوچک و بزرگ

کاربردهای پایتون

  • توسعه وب (Django, Flask)
  • هوش مصنوعی و یادگیری ماشین
  • تحلیل داده و علوم داده
  • اتوماسیون و اسکریپت‌نویسی
  • توسعه بازی (Pygame)

سرفصل‌های دوره:

  1. مقدمات پایتون: نصب و راه‌اندازی، متغیرها، انواع داده‌ها، عملگرها
  2. ساختمان داده‌ها: لیست، تاپل، مجموعه، دیکشنری
  3. کنترل جریان: شرط‌ها، حلقه‌های تکرار
  4. توابع: تعریف و فراخوانی، پارامترها، بازگشت
  5. ماژول‌ها و پکیج‌ها: import، کتابخانه استاندارد
  6. برنامه‌نویسی شی‌گرا: کلاس‌ها، اشیا، وراثت
  7. کار با فایل‌ها: خواندن و نوشتن فایل‌های متنی و باینری
  8. مدیریت خطا: try-except، ایجاد خطاهای سفارشی
  9. پایگاه داده: کار با SQLite و MySQL
  10. پروژه عملی: ساخت یک برنامه کاربردی کامل

ASP.NET یک چارچوب توسعه وب قدرتمند است که به شما امکان می‌دهد اپلیکیشن‌های وب مدرن و مقیاس‌پذیر بسازید.

مزایای ASP.NET

  • عملکرد بالا و امنیت قوی
  • پشتیبانی از الگوی MVC
  • توسعه سریع با ابزارهای Visual Studio
  • پشتیبانی مایکروسافت

کاربردها

  • اپلیکیشن‌های وب سازمانی

  • پورتال‌های اینترنتی
  • سیستم‌های مدیریت محتوای سفارشی

سرفصل‌های دوره:

  1. مقدمات ASP.NET: معماری، ابزارها

  5. وب API: ساخت سرویس‌های RESTful
  6. تست‌نویسی: Unit Testing
  7. پروژه عملی: ساخت یک اپلیکیشن وب کامل
